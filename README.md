# Homebrew Tap

Custom Homebrew tap for my tools and applications.

## Installation

```bash
brew tap barisercan/tap
```

## Available Packages

### Formulas (CLI tools)

| Formula | Description |
|---------|-------------|
| `arcsii` | Beautiful terminal-based code architecture visualizer with ASCII art |

```bash
brew install barisercan/tap/arcsii
```

### Casks (macOS apps)

| Cask | Description |
|------|-------------|
| `pane` | Fast keyboard-driven window switcher for macOS |

```bash
brew install --cask barisercan/tap/pane
```
